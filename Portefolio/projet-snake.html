Jeu Snake en C

Informations du projet

  • Date : Octobre 2024
  • Durée : 1 mois
  • Contexte : Projet universitaire
  • Technologies : C
  • Compétence : Développer des applications informatiques simples

Description

Développement d'un jeu Snake classique en langage C comme le vrai jeu du snake mais avec une limite de score de 10

Fonctionnalités

  • Déplacement du serpent avec les touches directionnelles
  • Génération aléatoire de nourriture
  • Augmentation de la taille du serpent lorsqu'il mange
  • Gestion des collisions avec les murs et le corps du serpent
  • Ajout d'obstacles sous forme de pavés

Défis techniques

Le principal défi a été pour commencer le faire bouger avec les flèches directionnelles puis la gestion d'apparaition des pavés.
