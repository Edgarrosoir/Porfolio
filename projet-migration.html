Installation de services réseau (SAÉ S2.03)

Informations du projet

  • Date : 2022/2023
  • Durée : 15 heures
  • Contexte : Projet universitaire
  • Compétence : Installer et configurer des services réseau

Description

Installation et configuration d'un serveur web complet sous Linux avec Apache, PHP et MySQL pour développer et déployer des applications web communicantes. Le projet répondait à un besoin client nécessitant un site dynamique avec des espaces d'accès sécurisés.

Tâches réalisées

  • Installation et configuration du serveur Apache
  • Mise en place du langage PHP côté serveur
  • Installation et sécurisation de la base de données MySQL
  • Création de pages publiques dynamiques (date, terminal client, etc.)
  • Mise en place d'un espace admin protégé par authentification Apache
  • Développement d'un système d'authentification via base de données
  • Création d'une page d'inscription avec validation par email
  • Personnalisation des pages d'erreur (404, etc.)
  • Configuration du nettoyage automatique des comptes non validés via cron

Technologies clés

  • Apache 2.4 (modules: mod_php, mod_ssl)
  • PHP 7.4 (configuration php.ini)
  • MySQL 8.0 (sécurisé avec mysql_secure_installation)
  • Authentification Basic (htpasswd)
  • Gestion des sessions PHP

Résultats

Serveur web fonctionnel avec :

  • Pages publiques dynamiques avec navigation
  • Espace admin sécurisé (login: admin, mot de passe: lannion)
  • Système d'inscription utilisateur avec validation
  • 3 comptes utilisateurs dans la base de données
  • Pages d'erreur personnalisées
  • Style CSS appliqué à toutes les pages
